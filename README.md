# My Workforce

Supervisor agents for OpenClaw. Each SOUL.md defines a category supervisor that can either handle tasks directly or spawn specialists from awesome-openclaw-agents templates.

## Usage

1. Copy desired supervisor SOUL.md into your OpenClaw `agents/` folder.
2. Add entry in `openclaw.json` under `agents.entries`.
3. Restart gateway.

Luna serves as the top-level coordinator.
